## Cara Pakai

5. Tap booth di peta untuk lihat detail creator

## Build

App ini dibuat menggunakan Flutter.

```bash
# Install dependencies
flutter pub get

# Run di browser
flutter run -d chrome

# Build untuk production
flutter build web --release
```
